Software Engineering Infrastructure

Describe the purpose of an Interactive Development Environment (IDE):
The purpose of an IDE is to provide a environment to build and test code/applications/websites and provide tools to help make that process faster and smoother.
Describe two advantages of an Interactive Development Environment:
Two advantages of an IDE are auto-formatting and linting features
Define the purpose of a Source control:
The purpose of source control is to give you the ability to access previous versions of your code without having to manually try to change your code back to a previous state and lose your current state of code. It solves the complexity of multiple developers working on the same file being able to update individual changes successfully.
Describe two advantages of utilizing a source control system to develop software:
Two advantages of using a a source control system are to save all versions of your code and to be able to share and access that code from anywhere.

Software Engineering Practice

What three programming languages are required to generate a web page?
HTML, CSS, Javascript
What is the purpose of writing pseudocode in software engineering?
Psuedocode is for writing out each step that is needed for your code to execute as planned. Then writing the necessary code in place of those steps.
Write pseudocode for a for loop that calculates the sum of all integers from 0 to 9:
  1. Change the data type to an array if needed.
  2. Declare a variable to store the sum of the integers.
  3. Create a loop that with iterate through each index of the array and take the value of each index and add it to the value of the "sum" variable.
  4. Return "sum" variable
Describe the purpose of a function in software engineering:
A function is a piece of code/object that is made to perform a specific task when called/invoked.
Describe the purpose of a library in software engineering:
A library is a set of pre-written and already tested code that can be used for things that you would have to create yourself. This helps speed up the development process and reduces some redundant coding.
